# J2DM - Binary Releases

This repository hosts compiled Windows binary releases of [J2DM](https://github.com/MNagasako/tool-jeoltif2dm),
a tool that converts JEOL SIGHTSKY TIFF images to Gatan DigitalMicrograph (.dm3 / .dm4) format.

Source code is kept in the private repository. Only the standalone .exe is published here.
